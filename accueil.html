REST API Pizzaland

Le projet "REST API Pizzaland" consiste à créer une API REST pour un service de livraison de pizza. L'API doit permettre de gérer les commandes, les pizzas et les ingrédients du pizzeria. L'API fait appel à une base de données PostgreSQL pour stocker les données et permet de réaliser des opérations de type GET, POST, PUT, PATCH et DELETE.
